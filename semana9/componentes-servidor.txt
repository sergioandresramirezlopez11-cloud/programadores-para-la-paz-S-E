Sistema operativo:
Es el software principal del computador. Administra los recursos del sistema y permite ejecutar aplicaciones.

Node.js:
Es un entorno de ejecución para JavaScript. Permite crear y ejecutar servidores fuera del navegador.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas y respuestas que recibirán los usuarios.

Puerto:
Es un punto de comunicación utilizado por el servidor para recibir solicitudes. En esta actividad se usa el puerto 3000.

Logs:
Son registros de eventos y actividades del sistema. Ayudan a monitorear el funcionamiento y detectar errores.
