El login permite identificar a un usuario antes de darle acceso a ciertas funciones de una aplicación. Algunas rutas deben estar protegidas porque contienen información privada o permiten realizar acciones importantes. Solo los usuarios autorizados deben acceder a ellas. No deben exponerse públicamente contraseñas, documentos de identidad, teléfonos, direcciones, tokens, claves de API ni otros datos sensibles de la comunidad.
