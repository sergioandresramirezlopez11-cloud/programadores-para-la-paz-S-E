Repositorio:
Un repositorio es un espacio donde se almacenan los archivos y cambios de un proyecto. Permite organizar la información y mantener un historial de todas las modificaciones realizadas.

Commit:
Un commit es un registro de cambios realizados en el proyecto. Cada commit guarda una versión específica para poder revisar o recuperar cambios anteriores.

Push:
El comando push se utiliza para enviar los cambios realizados desde el computador local hacia GitHub. Esto permite publicar y respaldar el proyecto en línea.
