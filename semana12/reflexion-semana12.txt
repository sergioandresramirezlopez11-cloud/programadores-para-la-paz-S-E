Durante este curso aprendí que la tecnología puede ser una herramienta muy importante para fortalecer la participación ciudadana y la organización comunitaria. Comprendí cómo integrar bases de datos, automatización y generación de documentos para desarrollar soluciones útiles para la comunidad. También entendí la importancia de proteger los datos personales y utilizar únicamente la información necesaria. Aprendí a trabajar con Git y GitHub para mantener un control de versiones y compartir proyectos de manera organizada. Además, conocí herramientas como Node.js y n8n, que permiten crear aplicaciones más eficientes. Este curso me permitió desarrollar habilidades que pueden contribuir a crear soluciones tecnológicas con impacto social y fortalecer la participación democrática.
